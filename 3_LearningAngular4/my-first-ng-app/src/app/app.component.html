Security settings. CORS - Cross Origin Resource Sharing - access control allow origin. Used to get around the browsers 'same origin' policy. For security purposes, a browser won't load requests for resources to other domains when those requests are initiated by JS. CORS supplies a special header that specifies which domains may make AJAX calls for its resources.
